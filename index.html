Setup Database
Koneksi ke Firebase — sekali saja
1
Buka console.firebase.google.com → Buat project baru (gratis)
2
Di menu kiri → Realtime Database → Create Database → pilih region → Start in test mode
3
Di tab Rules, paste ini lalu Publish: {"rules":{".read":true,".write":true}}
4
Copy Database URL (contoh): https://nama-project-rtdb.firebaseio.com
